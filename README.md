# Portfolio Website

A modern and responsive personal portfolio website built using React, Vite, Tailwind CSS, and Framer Motion.

Designed to professionally showcase projects, technical skills, education, resume, and contact information for recruiters, HR teams, internships, and professional opportunities.

---

## Live on Github
https://sahil-maurya-19.github.io/portfolio-website/

## 🚀 Features

- Modern Responsive UI
- Recruiter-Friendly Layout
- Animated Sections using Framer Motion
- Skills Showcase
- Project Cards
- Resume Download Support
- GitHub Pages Deployment Ready
- Clean React Component Structure
- Mobile-Friendly Design

---

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Sahil-Maurya-19/portfolio-website.git
```
Navigate into the project:
```
cd portfolio-website
```
Install dependencies:
```
npm install
```
Start development server:
```
npm run dev
```
Open:
```
http://localhost:5173
```
📦 Build for Production
```
npm run build
```
